R2018b

New Features, Bug Fixes, Compatibility Considerations
expand all

Firmware version update

Communications Toolbox™ Support Package for Analog Devices® ADALM-Pluto Radio now uses PlutoSDR version v0.28. For more information, see https://github.com/analogdevicesinc/plutosdr-fw/releases/tag/v0.28.

Burst mode reception: Pluto Receiver block and comm.SDRRxPluto System object can receive data from the ADALM-Pluto radio using burst mode

If the Pluto Receiver block or comm.SDRRxPluto System object™ cannot keep up with the ADALM-Pluto radio hardware, then your model or code is not processing data in real time. Burst mode enables you to buffer a set of contiguous samples without losing samples by setting the number of frames in a burst. For more information, see Burst Mode.

Built-in self-test: Run built-in-self-test of the ADALM-Pluto radio using the transmit and receive blocks and System objects

You can perform loopback and signal injection built-in self-tests (BIST) of the ADALM-Pluto radio via the Pluto Transmitter and Pluto Receiver blocks and the comm.SDRTxPluto and comm.SDRRxPluto System objects.

Direct digital synthesis: Directly synthesize additive tones on the ADALM-Pluto radio

You can use direct digital synthesis (DDS) to generate two additive tones on the ADALM-Pluto radio for transmission via the Pluto Transmitter block and the comm.SDRTxPluto System object™.

Functionality being changed

Compatibility Considerations

The following functionality has changed.

FunctionalityWhat Happens When You Use This Functionality?Use This InsteadCompatibility Considerations

The EnableQuadratureTracking, EnableRFDCTracking, or EnableBasebandDCTracking properties of comm.SDRRxPluto have been renamed.

If you enable or disable EnableQuadratureTracking are applied to EnableQuadratureCorrection and a warning is returned.

Use the EnableQuadratureCorrection property to enable or disable IQ quadrature correction.

The property names changed R2018b.

If you enable or disable EnableRFDCTracking are applied to EnableRFDCCorrection and a warning is returned.

Use the EnableRFDCCorrection property to enable or disable RF DC correction.

If you enable or disable EnableBasebandDCTracking are applied to EnableBasebandDCCorrection and a warning is returned.

Use the EnableBasebandDCCorrection property to enable or disable baseband DC correction.
